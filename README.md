# Hamiltonian Graph
This game has been made purely with HTML,CSS and Javascript.
# Description
•	A graph with number of vertices and edges are drawn on the canvas. We have to traverse the whole graph such a way that all nodes  are  traversed only once.<br/>
•	Starting of the game is from 0th node(Green colored ) of graph. <br/>
•	We can only make a move to the adjacent nodes of the current node. <br/>
•	Moving back is possible only from the currently visited node. <br/>
•	No of moves are counted on every click on nodes. <br/>
•	Timer is also displayed .(For this example, timer starts  from 45 minutes) <br/>
•	Submit button will get disabled after it is clicked once. Then you cannot click on any node. <br/>
